## 优先级 P0 · 正在开发

### ✅ 提示词助手（Prompt Enhancer）
- **目标**：降低新用户门槛，提升生成质量
- **实现**：在视频 / 图像 / 音乐输入框旁添加「优化」按钮，调用 LLM 将用户简短描述扩写成高质量英文提示词，同时给出 3 个变体供选择
- **涉及模块**：后端 `prompt` 模块、前端 `PromptEnhancer` 组件
- **完成标准**：按钮出现在所有创作页面、支持多语言、有加载态和变体选择 UI

## 优先级 P1 · 计划中（1-2 个月）

### 🔲 重试 / 微调历史
- 历史页每条记录加"基于此继续"按钮，将原始 prompt & 参数带回工作台
- 减少"一次不满意就放弃"的流失

### 🔲 批量生成 & 对比选优
- 图像生成支持同时输出 2-4 张不同风格结果（类 Midjourney Grid）
- 视频生成支持 2 个版本并排对比，用户选优后再精细化

### 🔲 嵌入 & 分享卡片
- 每个公开作品生成可嵌入的 iframe 播放器或精美 Open Graph 卡片
- 带品牌水印，社交分享时自带曝光

### 🔲 邀请裂变积分
- "邀请好友注册，双方各得 200 积分"
- 最低成本获客，经典裂变模型

---

## 优先级 P2 · 规划中（2-4 个月）

### 🔲 创作者主页 & 关注系统
- 每位用户有公开主页（作品集 / 粉丝数 / 擅长类型）
- 关注 / 粉丝订阅，把用户从"消费者"变"创作者"
- 参考：ArtStation + Civitai 模式

### 🔲 每日/每周创作挑战
- 官方发布主题挑战，社区投票，Top 3 获积分 / 徽章奖励
- 低成本高留存机制，Suno 等平台验证有效

### 🔲 作品 Collections / 合辑
- 用户把多个作品组合成合辑并分享链接
- 适合"AI 短片集"或"系列插画"创作者

### 🔲 图像 → 视频 工作台联动
- 图像页生成满意的图后，一键"发送到视频工作台"作为 i2v 起始帧
- 无需手动下载再上传

### 🔲 音乐 + 视频合成
- 生成 AI 音乐后，选择一段视频，系统自动 FFmpeg 合并
- 直接生成带背景乐的完整作品

## 优先级 P3 · 长期探索（4 个月以上）

### 🔲 提示词模板市场
- 官方 + 社区创作者提供场景化提示词模板
- 按场景分类（营销 / 故事 / 产品），可设"付费解锁"模板

### 🔲 创作者变现：付费作品 / 订阅
- 高质量创作者设置"积分购买完整版"
- 平台抽成形成正向飞轮

### 🔲 开发者 API & 控制台
- 面向开发者开放生成 API，积分计费
- 打开 B2B/B2D 市场，加一个 API 控制台页面

### 🔲 生成进度实时动画
- 生成中状态增加趣味动效（逐渐清晰 / 显示排队位置）
- 让等待体验更有趣

### 🔲 通知中心
- 别人点赞 / 评论你的作品、任务完成时有站内通知
- Bell icon 提升回访率
